# ESIKA BACKEND

This project served as API to a real estate web app

## PROJECT SETUP

```
npm install
```

### Compiles and hot-reloads for development

```
npm run dev
```

### Compiles and minifies for production

```
npm run build && npm start
```

## TECHNOLOGY

- TypeScript
- Node/Express
- MySQL
- TypeORM
